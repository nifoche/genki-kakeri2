# Kakeri2 - 掛け率計算機 Chrome拡張機能

定価、掛け率、販売価格率を入力して自動計算するChrome拡張機能です。

## 機能

- **定価**: 商品の定価を入力
- **掛け率**: 0.9 などの掛け率を入力（例: 0.9 = 90%）
- **販売価格率**: 80 などの販売価格率を入力（例: 80 = 80%）

自動計算で以下を算出：
- **基準価格**: 定価 × 掛け率
- **販売価格**: 基準価格 × 販売価格率

## インストール方法

### Chrome/Edge にインストール

1. このリポジトリをダウンロードまたはクローン
2. ブラウザで `chrome://extensions/` にアクセス
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `kakeri2` フォルダを選択

## 使用方法

1. ブラウザのツールバーにある拡張機能アイコンをクリック
2. 以下の値を入力：
   - 定価（例: 10000）
   - 掛け率（例: 0.9）
   - 販売価格率（例: 80）
3. 「計算」ボタンをクリック
4. 結果が表示されます

### 設定の保存・読み込み

- **現在の値を保存**: よく使う値を保存できます
- **保存した値を読み込み**: 保存した値を呼び出せます
- **設定をクリア**: 保存した設定を削除します
